UltraSmart One

A Multifunctional Device for Smart Urban Solutions — Mini Project, APJ Abdul Kalam Technological University (March 2026)

An integrated IoT-based smart urban management system that tackles three major city challenges — water wastage, inefficient parking, and improper waste disposal — in a single unified platform. The system uses an ESP32 microcontroller as the central processing unit, interfaced with ultrasonic (HC-SR04) and IR sensors for real-time automation.

The ESP32 is configured as a Wi-Fi Access Point hosting a local web dashboard, allowing real-time monitoring without any external internet dependency. Sensor data is transmitted in JSON format through an HTTP API endpoint and displayed dynamically in the browser.

💧

Water Management

Ultrasonic sensor monitors tank level. Pump AUTO ON when level drops below 8 cm threshold, OFF when tank is full — preventing overflow.

🚗

Smart Parking

IR sensor detects vehicles at entrance. Servo motor controls gate. Two ultrasonic sensors monitor slot occupancy with LED status indicators.

🗑️

Waste Management

Ultrasonic sensor tracks garbage bin level. Buzzer alert triggers when bin reaches capacity threshold — ensuring timely collection.

Hardware used: ESP32, HC-SR04 ultrasonic sensors, IR sensor, servo motor, relay module, BC547 transistor, LEDs, buzzer, water pump, perfboard. Software: Arduino IDE with WiFi.softAP(), millis()-based non-blocking operation.
